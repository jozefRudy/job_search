When copying UI components from another project, always verify: 1) missing dependencies (e.g., `ts-pattern`), 2) missing static assets (e.g., `icons.svg`), 3) theme alignment (`data-theme` attribute vs daisyui config), 4) `~` alias resolution in vite config.
